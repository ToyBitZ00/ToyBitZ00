## 🚀 Featured Projects

### ☕ Kape't Bahay — Café Sales Management & Ordering System
> **Final Project · Software Engineering 01**

A full-featured **café point-of-sale and management system** built in Python with CustomTkinter for the UI and SQLite for the database backend. Designed to simulate a real client-based software delivery workflow, covering requirements gathering, UI/UX design, debugging, and optimization.

**Key Features:**
- 🔐 Secure login & credential management
- 🛒 Interactive cart & order placement (size, quantity, subtotal)
- 📋 Live order queue with status tracking (pending → preparing)
- ☕ Beverage management (add items, update prices, toggle availability)
- 📊 Sales reporting: daily/weekly revenue, order counts, popular items

**Tech:** `Python` · `CustomTkinter` · `SQLite`  
**Commits:** 67 · **Team project with multi-developer collaboration**

[![View Repo](https://img.shields.io/badge/View_Repository-181717?style=for-the-badge&logo=github)](https://github.com/ToyBitZ00/Kape-t_Bahay_Cafe_Sales_Management_And_Ordering_System)

### 🚨 Community Concern & Response System
> **Final Project · Applications Development · 3rd Year, 1st Semester**

A desktop application built to help **local government officials in San Miguel, Bulacan, Philippines** track and respond to calamity reports submitted by community residents. Tackles a real-world civic problem with a practical software solution.

**Key Features:**
- 📍 Residents can file and submit community concerns
- 🏛️ Official dashboard to view, prioritize, and respond to reports
- ✅ Resolved concern tracking with image attachments
- 🔐 Admin authentication system

**Tech:** `Python` · `Tkinter` · `File-based data storage`  
**Impact:** Designed for real local government use
